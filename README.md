======================================================
EEPlat则是一种传统独立部署模式的企业应用快速开发应用平台。  
EEPlat PaaS平台是一种面向企业应用领域，以元数据驱动为核心架构思想，能够支持多租户应用的在线快速软件开发、系统部署及运行PaaS平台。
